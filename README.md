## Description

**Network City Helper** - персональный телеграм бот созданный для помощи ученикам в учебе. Он может показывать итоговые оценки, уведомлять о просроченных заданиях, отправлять расписание, д/з на завтра и многое другое!
Кроме того, функциональность бота легко расширять или ограничивать по вашим потребностям.

_Предоставленный код находится в недоработанном виде и возможно требует доработки. В нем присутствует избыточный код, и его можно оптимизировать для улучшения производительности и читаемости._
